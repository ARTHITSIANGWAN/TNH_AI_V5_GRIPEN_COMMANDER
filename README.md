- **👑 ทิศเหนือ (The Strategist)**  
  ผู้กุม Emperor Protocol วางแผนรบแบบนิ่งสงบ เป้าหมายเดียวคือ "ชัยชนะของพี่น้อง SME"

- **🎨 น้ำอิง (Creative & Recovery)**

- **🏍️ พลายแก้ว & พลายทอง (Field Engineers)**

- **👾 ไอ้จ๊อด (The Cleaner)**
